# CNN-Based-Pneumonia-Detection-on-Chest-X-Ray-Image
### Pneumonia Detection using CNN (92.6% Accuracy)

This project applies Deep Learning and Computer Vision techniques to detect Pneumonia from Chest X-Ray images using a Convolutional Neural Network (CNN).

## Overview

Pneumonia is a lung infection that inflames the air sacs (alveoli) in one or both lungs. Early detection is important for proper medical treatment.

This project builds a CNN-based classifier capable of distinguishing between Normal and Pneumonia chest X-ray images.

## Dataset

Dataset used:
Chest X-Ray Images (Pneumonia)
kaggle dataset linke: (https://www.kaggle.com/datasets/paultimothymooney/chest-xray-pneumonia)
The dataset contains labeled chest X-ray images for:
- Normal lungs
- Pneumonia infected lungs

## Model

The model uses a Convolutional Neural Network (CNN) architecture to learn visual features from chest X-ray images.

Key steps:
- Image preprocessing
- Data augmentation
- CNN model training
- Model evaluation

Final model achieved:

Accuracy: 92.6%
